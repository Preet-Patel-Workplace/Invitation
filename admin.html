How to receive RSVPs via email & see them in Formspree

  1. Go to formspree.io → Sign up free → Click "New Form"
  2. Name it "Baby Shower RSVP" → Copy the endpoint ID (e.g. xpwzabcd)
  3. Open js/main.js → Find line with YOUR_FORM_ID → Paste your ID
  4. Redeploy to Vercel → Every RSVP emails you instantly + logs to Formspree dashboard

📊 Formspree dashboard: formspree.io/forms → click your form → see all submissions in a table, export CSV
